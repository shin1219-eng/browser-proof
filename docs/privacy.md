# Privacy Policy

The Browser Proof System is designed with privacy-first principles.

## Data Collection
- **Verification Input**: We collect the source URL and the specific task description.
- **Verification Output**: We store the extracted evidence (text) and a screenshot for audit and verification purposes.
- **Access Logs**: Standard HTTP access logs for security and performance monitoring.

## Third-Party Processors
- **Browserbase**: Provides the secure browser infrastructure.
- **OpenAI**: Used for the semantic extraction and verification logic.
- **Supabase**: Handles the persistent storage of logs and evidence.

## Guidelines
- **Public Content Only**: Do NOT use this service to verify private, personal, or authenticated content.
- **De-identification**: Ensure that your verification tasks do not include personal identifiable information (PII).
